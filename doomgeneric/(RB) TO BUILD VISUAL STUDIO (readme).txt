DoomGeneric in Visual Studio, you can add references to files in the Source and Text Screen folders, and then build the project. 
Steps 
Open the project in Solution Explorer
Right-click on the project
Select Add then Existing Item
Select the files to add from the Source and Text Screen folders
Select Build from the menu bar
Select Build ProjectName to build only the components that have changed since the last build
You can find the doomgeneric.sln file on GitHub. 
Additional information
You can also build Chocolate Doom on Windows using Visual Studio, but MSYS2 is the "canonical" way to build it. 
You can also build Doom for the Pico 2 on a Windows machine without using Visual Studio.
